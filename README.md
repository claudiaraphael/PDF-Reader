# Personal PDF Reader

This is a desktop based application that reads PDFs and allows you to mark the files. I am developing it for my own use and I intend to distribute this software as open-source. This project uses the following technologies:

🛠️ Stack 
- Python 3.10+

- GUI Framework: PySide6 (Versão oficial do Qt para Python).

- PDF Engine: PyMuPDF (pip install pymupdf).

- Packaging (Transformar em .exe): PyInstaller ou Nuitka.
